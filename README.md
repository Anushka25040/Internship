# Software Testing Internship Task

## Overview
This repository contains the tasks completed as part of the Software Testing Internship program. It includes test cases, bug reports, test scenarios, and related documentation created during the internship.

## Objectives
- Understand software testing concepts
- Design and execute test cases
- Identify and report defects
- Perform functional and non-functional testing
- Document testing activities

## Contents
- Test Cases
- Test Scenarios
- Bug Reports
- Test Execution Reports
- Supporting Documents

## Tools Used
- Manual Testing
- Microsoft Excel / Google Sheets
- JIRA (if applicable)
- Git & GitHub
